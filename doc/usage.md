# AutoDub 使用教程

欢迎使用 AutoDub 二创工具！本工具通过 AI 技术赋能，帮助短视频创作者提高内容制作效率。

## 版本说明

AutoDub 提供三个版本：

- **免费版**：基础功能，适合入门用户。每月150条短视频生成额度
- **基础版**：增强功能，支持更多自定义选项。每月600条短视频生成额度
- **高阶版**：完整功能，支持高级特性。短视频生成额度不限制

## 安装

1. 从百度网盘或 GitHub 下载软件安装包
2. 解压压缩包（无需安装）
3. 双击运行 `AutoDub.exe` 启动程序

## 申请许可证书

首次使用需要申请许可证书，步骤如下：

1. **获取设备标识**
   - 首次运行程序时，系统会自动获取设备的唯一标识
   - 设备标识界面如下图所示：

   ![设备唯一标识](/resource/deviceUid.jpg)

2. **申请许可证书**
   - 复制设备标识
   - 进入 QQ 群申请许可证书（请提供设备标识）

3. **配置许可证书**
   - 将获得的许可证书文件放置到 `_internal` 文件夹下
   - 再次运行软件即可正常使用

## 核心功能

### 1. 新建任务

在新建任务界面，您可以配置视频处理任务的各种参数。

![新建任务界面](resource/add_task.jpg)

#### 任务配置说明

- **TTS 服务**
  - 免费版：仅可使用内置的 Edge 音色
  - 基础版：可自行配置火山引擎语音合成大模型的音色
  - 高阶版：可免费获取本地音色服务 API 脚本（安装需额外付费）

- **翻译服务**
  - 统一使用火山引擎大模型
  - 后续将持续支持阿里、硅基流动等服务商

- **人声分离**
  - 启用后会自动分解出原视频的 BGM（背景音乐）
  - 免费版不可用，基础版及高阶版可用

- **启用多说话人区分**
  - 启用后自动识别视频中的多个说话人
  - 免费版及基础版不可用，仅高阶版可用（需部署本地 TTS 服务）

- **仅添加字幕**
  - 适用于英文流视频，仅添加字幕，不进行翻译
  - 此功能仅对非免费用户开放

- **仅下载视频**
  - 不做任何处理，仅下载视频文件

### 2. 调度中心

在调度中心界面，您可以查看和管理所有任务。

![调度中心](resource/queue.jpg)

**功能说明**：

- 查看所有任务信息（执行成功的任务不再显示）
- 点击执行失败的任务可查看详细失败原因
- 实时监控任务执行状态

### 3. TTS 资源管理

基础版和高阶版用户可在此界面添加和管理火山引擎的音色资源。

![TTS资源界面](resource/tts.jpg)

#### 添加音色资源

点击"新增资源"按钮，在表单中填写以下参数：

| 参数 | 说明 | 是否必填 |
| --- | --- | --- |
| 服务商 | 选择 `volcengine` | 必填 |
| base_url | `https://openspeech.bytedance.com` | 必填 |
| access token | 从火山引擎控制台获取 | 必填 |
| secret key | 无需填写 | 否 |
| appid | 应用 ID | 必填 |
| resourceID | 音色资源 ID，建议选择第一个 | 必填 |
| 音色名称 | 自定义音色名称 | 必填 |

#### 获取配置信息

- **获取 appid 和 access token**：
  - 访问：[火山引擎 TTS 控制台](https://console.volcengine.com/ark/region:ark+cn-beijing/tts/speechSynthesis)

- **查看可用音色列表**：
  - 访问：[音色列表文档](https://www.volcengine.com/docs/6561/1257544?lang=zh)

### 4. 翻译资源管理

所有用户均可配置翻译资源。目前支持火山引擎大模型。

![翻译资源界面](resource/translate.jpg)

#### 配置参数

| 参数 | 说明 | 是否必填 |
| --- | --- | --- |
| 服务商 | 选择 `volcengine` | 必填 |
| base url | `https://ark.cn-beijing.volces.com/api/v3` | 必填 |
| api key | API 密钥 | 必填 |
| appid | 无需填写 | 否 |
| model | 选择翻译模型 | 必填 |
| temperature | 温度参数（如不了解，保持默认即可） | 可选 |

#### 获取 API Key

访问火山引擎控制台获取 API Key：
[API Key 获取页面](https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey?apikey=%7B%7D)

### 5. 上传到 Bilibili

此功能仅对非免费用户开放，帮助用户快速上传作品到 B 站。

![上传界面](resource/upload.jpg)

#### 功能特性

- 自动生成标签
- 自动选择分区
- 自动生成封面图（含 TikTok 封面图）
- 自动生成标题
- 一键上传

#### 字体配置

本项目不提供商业可用的字体，请自行购买字体，并将字体路径按照 `fonts` 文件夹下的 JSON 数据格式填写。

## 设置

在设置界面，您可以配置以下选项：

- **结果存储目录**：设置所有生成结果的存储路径
  - 所有处理完成的视频、字幕等文件均保存在此目录
  - 请定期清理，避免磁盘空间不足

## 常见问题

### Q: 如何查看任务执行失败的原因？

A: 在调度中心界面，点击执行失败的任务即可查看详细错误信息。

### Q: 免费版和付费版有什么区别？

A: 主要区别在于功能限制：

- 免费版：仅支持基础 TTS 和翻译功能
- 基础版：支持人声分离、自定义 TTS 音色等
- 高阶版：支持多说话人区分等高级功能

### Q: 如何获取火山引擎的 API 密钥？

A: 请访问火山引擎控制台，按照上述文档中的链接获取相应的 API Key 和 Access Token。
